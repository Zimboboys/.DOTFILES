# `nix`

sustainable configurations for packages to help keep the environment clean.

## home manager

most other dotfiles and profile settings are managed with [home manager][hm].

```
$ ./nix.sh setup
$ home-manager switch
$ cowsay darn tootin
 _____________
< darn tootin >
 -------------
        \   ^__^
         \  (oo)\_______
            (__)\       )\/\
                ||----w |
                ||     ||
```
