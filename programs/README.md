# `programs`

bundled configurations for packages to help keep the environment clean.

## home manager

most dotfiles and profile settings are managed with [home manager][hm].

```
$ source setup.sh run
$ home-manager switch
$ cowsay darn tootin
 _____________
< darn tootin >
 -------------
        \   ^__^
         \  (oo)\_______
            (__)\       )\/\
                ||----w |
                ||     ||
```
